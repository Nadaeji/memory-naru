# 기억 나루 🌿  
**감성 회상 기반 인지 자극 게임**

---

## 🧠 프로젝트 소개

'기억 나루'는 **경도인지장애 및 초기 치매 환자**를 위해 설계된  
감성 회상 기반의 인지 치료 도구입니다.  
이 프로젝트는 단순한 기억력 게임이 아닌,  
**사용자와 기억, 감정, 존재를 다시 연결해주는 여정**입니다.

‘나루’라는 친구형 AI가  
사용자와 함께 기억을 떠올리고, 감정을 나누며,  
치료자나 보호자보다 더 가깝고 편안한 **정서적 말동무**로 존재합니다.

---

## 🎯 기획 목표 및 핵심 고려 사항

### 🌿 1. **치료 목적의 감성 설계**
- 단순한 문답형 훈련이 아닌 **정서적으로 따뜻한 회상 환경 제공**
- AI를 감지하지 못할 만큼 자연스러운 대화 흐름 (반말, 공감적 화법)
- 기억 회상이 **억지스럽지 않고, 편안하고, 스스로 떠오르게** 설계

### 🧭 2. **기억 흐름 구조**
- 회상의 범위를 최근 → 중간 → 오래전 기억으로 점진적 확장
- 초기엔 '오늘', '이번 주', '최근 명절'처럼 부담 없는 회상부터 시작
- 사용자의 기억 접근이 용이하도록 **순차적이고 감정적으로 유도**

### 💡 3. **감정 표현과 기록**
- 사용자가 감정을 직접 선택하도록 유도하여 **감정 회상도 병행**
- 감정은 회상 기억보다 선명히 남아 있는 경우가 많다는 점을 고려
- 감정 태그는 사용자 맞춤형으로 확장 가능 (복잡함, 불안함 등 포함)

### 🪄 4. **AI의 ‘친구화’ 전략**
- “나는 AI야”라고 밝히지 않음 → 사용자에게는 ‘나루’라는 친구
- '공감적 반말'을 통해 **진짜 대화**처럼 느껴지도록 설계
- 과거 기억을 함께 걷는 듯한 **감정적 동반자 역할** 수행

### 🔁 5. **장기 사용 가능성**
- 회상 흐름은 순환 가능하도록 설계 → 매일 반복 사용해도 새로움 유지
- 감정 기록은 날짜별로 저장되어 **감정 변화 추이 시각화 가능**
- 사용자 상태 변화에 따른 회상 수준 조절 및 기억 맞춤 질문도 고려

## 💡 기능 개요

| 기능 | 설명 |
|------|------|
| 🌿 기억 회상 흐름 | 최근 → 중간 → 오래된 기억 순으로 질문 생성 |
| 💬 친구형 AI 응답 | 감정 공감형 반말 화법, AI임을 밝히지 않음 |
| 🧠 감정 선택 | 감정 이모지와 라벨로 사용자 감정 기록 유도 |
| 📊 감정 리포트 | 회상 기록과 감정 기록을 날짜별 자동 저장 |
| 📈 장기 변화 추적 | 감정 리포트를 바탕으로 장기 추이 시각화 가능 |
| 🖥 감성 UI | Streamlit 기반 웹 앱으로 감성적 디자인 제공 |

## 📊 사회적 배경 및 필요성

### 치매 및 경도인지장애의 증가 추세

- **치매 환자 수**: 2025년 97만 명(유병률 9.17%) → 2044년 201만 명 예상  
- **경도인지장애 진단자 수**: 2025년 298만 명(유병률 28.12%) → 2033년 400만 명 예상  
  > 출처: [보건복지부 치매관리종합계획(2023)](https://www.mohw.go.kr/board.es?mid=a10503010100&bid=0027&act=view&list_no=1484959)

### 경도인지장애에서 치매로의 진행

- **진행률**: 경도인지장애 노인의 약 21%가 2년 이내 치매로 진행됨  
- **회귀율**: 사회적 교류가 활발한 경우 인지기능 정상 회귀 가능성 증가  
  > 출처: [DBpia 치매노인의 사회적 특성 연구](https://www.dbpia.co.kr/journal/detail?nodeId=T15937587)

### 회상기법의 효과

- **인지기능 향상**: 회상 훈련은 알츠하이머형 치매노인의 인지기능 향상에 효과  
- **우울감소**: 회상치료는 우울감과 기억력 저하를 완화하는 데 효과적  
  > 출처: [DBpia 회상치료 메타분석 논문](https://www.dbpia.co.kr/journal/articleDetail?nodeId=NODE11125205)

### 사회적 교류의 중요성

- **우울 유병률**: 경도인지장애 노인의 약 33.2%가 우울 증상 경험  
- **말동무의 필요성**: 친밀하고 지속적인 대화는 인지 자극과 정서 안정에 효과적  
  > 출처: [한국노인간호학회지](https://j.kafn.or.kr/journal/view.php?number=1104)

## 💡 프로젝트 목적

- **인지 기능 자극**: 최근 → 과거 흐름을 따라 질문하며 자연스럽게 회상을 유도
- **감정 자각과 공유**: 회상한 기억에 감정을 덧붙이며 감성적 자극 및 치유 도모
- **장기 추적 기반 데이터 저장**: 감정 리포트 및 회상 내용 로그 기록
- **친근한 말투와 존재**: AI가 아닌 친구 같은 말동무로 인식되도록 설계
